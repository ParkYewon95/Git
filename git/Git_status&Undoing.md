## status

1. 새로 파일 생성 한 경우

``` bash
$ git status
On branch master

No commits yet

#commit 이력에 담긴 적 없는 파일들
```

2. add 한 이후

```bash
$ git add a.txt
$ git status

No commits yet
#커밋될 변경사항들(changes)
Changes to be committed:
  (use "git rm --cached <file>..." to unstage)
        new file:   a.txt

```
